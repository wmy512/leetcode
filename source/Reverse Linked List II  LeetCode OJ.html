Reverse Linked List II

Reverse a linked list from position m to n. Do it in-place and in one-pass.

For example:
Given 1->2->3->4->5->NULL, m = 2 and n = 4,

return 1->4->3->2->5->NULL.

Note:
Given m, n satisfy the following condition:
1 ≤ m ≤ n ≤ length of list.

/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
public:
    ListNode *reverseBetween(ListNode * head, int m, int n) {
        // Note: The Solution object is instantiated only once and is reused by each test case.
        // 先将要转换的部分按头节点为空的构建链表，保持第一个和最后的节点，以及转换后链表的最前最后的节点，最后连接起来。
        // 第一次出现错误，当要转换的m大于1时，原头指针不变。
        if ( head == NULL) {
            return NULL;
        }
        if ( m == n) {
            return head;
        }
        ListNode *pre = NULL;
        ListNode *cur = head;
        ListNode *reverse_head = NULL;
        ListNode *first_head = NULL;
       
        int index = 1;
        while (index < m) {
            pre = cur;
            cur = cur->next;
            index++;
        }
        first_head = pre;
        pre = NULL;

        while (index <= n) {
            if (pre == NULL) {
                reverse_head = cur;
            }
            ListNode *next = cur->next;
            cur->next = pre;
            pre = cur;
            cur = next;
            index++;
        }
        //restore the whole link
        if (first_head) {
            first_head->next = pre;
        }
        else {
            first_head = pre;
        }
        if (reverse_head) {
            reverse_head->next = cur;
        }
        if ( m > 1) {
            return head;
        }
        else {
            return first_head;
        }
    }
};
